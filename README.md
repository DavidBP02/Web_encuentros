# 🍺 Complejo Hostelero Encuentros - Sitio Web

Una web moderna y completa para el Complejo Hostelero Encuentros de La Pueblanueva, Toledo.

## 📁 Estructura del Proyecto

```
web_encuentros/
├── index.html          # Página principal
├── css/
│   └── styles.css     # Estilos CSS
├── js/
│   └── script.js      # Funcionalidad JavaScript
└── README.md          # Este archivo
```

## 🚀 Características

✅ **Diseño Responsive** - Se adapta a cualquier dispositivo (móvil, tablet, desktop)
✅ **Navegación Intuitiva** - Menú sticky con smooth scrolling
✅ **Menú Hamburguesa** - Para dispositivos móviles
✅ **Secciones Completas:**
   - Inicio con hero image
   - Historia del negocio
   - Pub con servicios
   - Restaurante con menús y especialidades
   - Terraza de verano
   - Eventos y celebraciones
   - Galería de fotos
   - Contacto con formulario y mapa
